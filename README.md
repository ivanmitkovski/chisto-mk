# chisto.mk

Civic mobile application enabling citizens in Macedonia to report illegal dumping, pollution, and environmental hazards.

## Monorepo structure
- `mobile/` Flutter app
- `backend/` Node.js API (planned)
- `web/` Next.js landing + admin (planned)
- `docs/` technical documentation and ADRs
- `infra/` deployment and infrastructure (later)

## Getting started (mobile)
```bash
cd mobile/chisto_mobile
flutter pub get
flutter run
